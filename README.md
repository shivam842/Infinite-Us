# The Infinite Us - An Atmospheric Connection Environment

An intimate, ethereal Android application built with Jetpack Compose and AGSL Shaders.

## Features
- **Nebula Engine**: A generative cosmic background using AGSL RuntimeShaders.
- **Presence-First UI**: Zero-UI design focusing on emotional connection.
- **Real-time Sync**: Firebase Realtime Database integration for partner presence and pulse.
- **The Eclipse**: A visual and haptic event triggered when users pulse in sync.

## Tech Stack
- **Kotlin DSL** for Gradle
- **Jetpack Compose** for UI
- **AGSL** (Android Graphics Shading Language)
- **Firebase Realtime Database**

## Setup
1. Add your `google-services.json` to the `app/` directory.
2. Enable Firebase Realtime Database in your Firebase Console.
3. Build and run on an Android 13+ (API 33) device.
